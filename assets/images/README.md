# Images Directory

Place your image assets here.

## Recommended Structure

```
assets/images/
├── icons/          # App icons and small graphics
├── logos/          # App logos and branding
├── backgrounds/    # Background images
└── placeholders/   # Placeholder images
```

## Image Guidelines

- Use WebP format for better compression
- Provide 1x, 2x, 3x versions for different screen densities
- Optimize images before adding to the project
- Use descriptive names (e.g., `icon_home.png`, `logo_app.png`)

## Example

```dart
Image.asset('assets/images/logos/logo_app.png')
```
